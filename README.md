# The Sims 4 Tools

A big set ot methods and tools which I use for my mods.

Please don't use it for your mods. There are other, better documented and more stable libraries available.

## Compatibility

This mod is not compatible with [S4CL](https://github.com/ColonolNutty/Sims4CommunityLibrary).

#### TS4 Mods hosted on MTS
Most mods from MTS require this library (or S4CL).

If S4CL is installed they will use the S4CL API instead of this mod. Don't install both mods as this will break the game!
If you have S4CL installed for other mods just keep it and skip the installation of this mod.

#### TS4 Mods hosted on GitHub
These mods usually require S4CL. If you are using one of them avoid installing TS4T.

# The Sims 4 Library

A big library with many methods I use for my mods.

Please don't use it for your mods. There are other, better documented and more stable libraries available.

## Compatibility

This mod is not compatible with [S4CL](https://github.com/ColonolNutty/Sims4CommunityLibrary).

If S4CL is available all of my mods will use the S4CL API instead of this mod. Don't install this mod if S4CL is installed.
